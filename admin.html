Admin Login

Sign in with a GitHub fine-grained access token that has access only to the website repository. The token is kept only in this browser tab and is never added to the website files.

How to create the correct token
  1. Open GitHub Settings → Developer settings → Personal access tokens → Fine-grained tokens.
  2. Grant access only to JeffFoldsHomeImprovement.github.io.
  3. Set Repository permissions → Contents to Read and write.
  4. Copy the token and paste it above. Do not email or share the token.
