JEFF FOLDS' HOME IMPROVEMENT WEBSITE

HOW TO OPEN
1. Unzip the website folder.
2. Double-click index.html to preview it in a browser.

HOW TO SET THE BUSINESS EMAIL
1. Open config.js in Notepad or another text editor.
2. Replace CHANGE-ME@example.com with the new business email address.
3. Save the file.
4. The form uses FormSubmit.co. The first form submission will cause FormSubmit to send a verification message to the business email. Approve it once, and future requests will be forwarded there.
